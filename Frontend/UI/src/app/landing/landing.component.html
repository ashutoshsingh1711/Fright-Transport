SERVICES

FTR Facilities

Freight Transport Region application helps to manage the overall operation in Harbor.
Admin(Employees of FTR) and User(Vendor) can avail this application.

Admin Services

FTR Admin can manage the major details like
managing,loading and grouding operations,
drivers mapping for each vehicle, garbage collection and deallocation, permit barrier access for the vehicle and managing sensor details etc.

Customer Services

Customers can register with FTR and request for freight transport, includes the various user specific configurations
and it can be viewed using Operational Dashboard
